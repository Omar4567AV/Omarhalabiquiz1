# 🚀 React Project (Vite + TypeScript)

This project was created using **Vite**, **React**, and **TypeScript**, then uploaded to GitHub.

---

## 📌 Project Setup

Create a new React project using Vite:

```bash
npm create vite@latest my-app
```

Select:

* Framework: **React**
* Variant: **TypeScript**

---

## 📂 Open the Project

```bash
cd my-app
code .
```

---

## 📦 Install Dependencies

```bash
npm install
```

---

## ▶️ Run the Project

```bash
npm run dev
```

Open the link shown in the terminal (usually http://localhost:5173).

---

## 🔗 First Upload to GitHub

Initialize Git and push your project:

```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Omar4567AV/Omarhalabiquiz1.git
git push -u origin main
```

---

## 🔄 Update Project on GitHub (VERY IMPORTANT)

After making changes to your project, use these steps to update your repository:

```bash
git add .
git commit -m "update project"
git push
```

### 🧠 Explanation:

* `git add .` → adds all new changes
* `git commit -m "..."` → saves changes with a message
* `git push` → uploads changes to GitHub
